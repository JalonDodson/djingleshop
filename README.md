## Installation

Use the dependency manager [poetry](https://github.com/python-poetry/poetry) to install.

POETRY: 
```bash
git clone git@github.com:JalonDodson/djingleshop.git
cd djingleshop
poetry install
poetry shell
python manage.py runserver
```

PIP: 
```bash
git clone git@github.com:JalonDodson/djingleshop.git
cd djingleshop
pip install -r requirements.txt
python manage.py runserver
```
